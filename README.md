### Prerequisites

In order to run this project you need:
- Node.js

### Setup

Clone this repository to your desired folder:

```bash
  git clone git@github.com:OsvaldoBC/hello-react-frontend.git
  cd hello-react-front-end
```

Install this project with:

  `npm install`
